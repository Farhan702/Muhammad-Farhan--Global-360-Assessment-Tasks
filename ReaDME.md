# Snipe-IT Playwright Automation (.NET)

This project is a single-file end-to-end automation test using .NET and Playwright.

## Tech Stack
- .NET 8
- Microsoft Playwright
- C#

## Test Flow
1. Login to Snipe-IT demo
2. Create a MacBook Pro 13" asset
3. Set status to Ready to Deploy
4. Assign asset to a user
5. Verify asset in list
6. Validate asset details
7. Validate History tab
